# OOTP 27 KBO Photopack

![OOTP 27 KBO Photopack](./previews/1.png)

OOTP 27용 KBO 선수 사진팩입니다. 파일명은 `bbrefminors_id` 기준으로 정리되어 있습니다.

## 설치 방법

![설치 방법](./previews/2.png)

1. `photos` 폴더 안의 PNG 파일을 OOTP 27의 `data/photos` 폴더에 복사합니다.
2. 기존 파일이 있으면 덮어씁니다.
3. 게임에서 `Settings -> FaceGen -> Picture Search Mode -> Search for photos first` 로 설정합니다.
4. `Update/Generate All Player Pictures` 를 실행합니다.

예시:

- Windows (Steam): `...\\Out of the Park Baseball 27\\data\\photos`

## 포함 내용

- 선수 사진 621장
- 이미지 크기: `90x135`
- 파일명 형식: `bbrefminors_id`
